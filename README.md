## 基本環境建置流程
1. 安裝NodeJs
⇒　[官方網站](https://nodejs.org/zh-tw/)
* 注意要安裝LTS的版本！
* 安裝完成試著執行node -v，如有問題可以試試重新登出再登入或重開機。
* npm -v如果可以執行就表示安裝完成。

2. 安裝@angular/cli
⇒npm i -g @angular/cli

3. 如果安裝成功可以測試使用ng v查詢版本，或使用ng new建立新專案。

3. 安裝VS Code⇒[下載位置](https://code.visualstudio.com/download)
